Relate API Documentation

0. Basics

Authorization:

Relate API uses Basic HTTP Authorization.
You need just add Header Authorization with base64 encoded login and password.

Exaple:

Authorization: Basic QWxhZGRpbjpPcGVuU2VzYW1l

URLs:

  • DEV: https://dev-relate247-api.realpage.com/api/
  • PROD: https://relate-api.realpage.com/api/

PHP Api setup

Check
Relate\Configuration::_construct()
$this->setClientId('CLIENTID');
$this->setHost('APIURL');
$this->setUsername('RELATEUSER');
$this->setPassword('RELATE_PASS');

1. Get All properties

To retreive data you should know your RELATE_CLIENT_ID.

Use HTTP Get request to /property/RELATE_CLIENT_ID.

Example:

GET /api/property/1234 HTTP/1.1
Host: dev-relate247-api.realpage.com
Content-Type: application/json
Authorization: Basic QWxhZGRpbjpPcGVuU2VzYW1l

-

<?php
    $propertyApi = new Relate\Api\PropertyApi();
    $property = $propertyApi->getAllProperties(); // Look inside function to find RELATE_CLIENT_ID

This will return list of properties assigned to your CLIENT_ID. Use PROPERTY_ID in all next requests by specifying it in Header pid.

pid: PROPERTY_ID

2. Create/Add Prospect

Use POST request to /prospect with JSON body:

  • firstname: Prospect's First Name
  • lastname: Prospect's Last Name
  • emailaddress: Prospect's Email Address

Example:

POST /api/prospect HTTP/1.1
Host: dev-relate247-api.realpage.com
Content-Type: application/json
pid: 1005544
Authorization: Basic QWxhZGRpbjpPcGVuU2VzYW1l
{"firstname":"Ryan","lastname":"Ma","emailaddress":"ryan.ma@example.net"}

-

<?php
    $propertyApi = new Relate\Api\PropertyApi();
    $prospect = new Relate\Model\Prospect('Ryan', 'Ma', 'ryan.ma@example.net');
    $prospectApi->addProspect($prospect);

4. Get Prospect

Identification of prospects are done with their emails.

Use HTTP Get request to /property/PROSPECT_EMAIL.

Example:

GET /api/prospect/ryan.ma@example.net/ HTTP/1.1
Host: dev-relate247-api.realpage.com
Content-Type: application/json
pid: 1005544
Authorization: Basic QWxhZGRpbjpPcGVuU2VzYW1l

-

<?php
    $propertyApi = new Relate\Api\PropertyApi();
    $prospect = $prospectApi->getProspect('ryan.ma@example.net');
